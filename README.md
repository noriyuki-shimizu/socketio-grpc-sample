# socketio-grpc-sample

Realtime AI chat sample that streams Gemini Pro Vision responses into a Nuxt 4 UI via Socket.io and a local gRPC gateway.

## Dev scripts

```bash
npm run dev        # start server + web in parallel
npm run dev:server # backend only
npm run dev:web    # frontend only
```

Copy `.env.sample` to `.env` and fill the secrets before running the backend.
